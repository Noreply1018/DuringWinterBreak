# 影像裁剪小工具

通过 conda 搭建 gdal 专属环境，安装 gdal 库并验证成功，配置中文路径与静默警告。学习 GDAL 核心流程：注册驱动 - 打开影像 - 获取元信息 - 处理 - 保存，掌握 Dataset/Band 核心对象，完成影像信息查看、裁剪小工具编写，理解地理变换与投影的空间关联，后续将继续学习重采样与批量处理。

聚焦 GDAL 影像裁剪小工具开发，先复配 GDAL 环境并验证可用，配置中文路径避免文件读取报错。梳理工具核心逻辑：影像打开 - 坐标转换 - 范围校验 - 数据读写 - 结果保存，完成基础框架编写，实现地理 / 像素坐标的基础解析，测试单波段 TIFF 影像可正常打开并提取地理变换参数。

完善影像裁剪小工具核心功能，新增裁剪范围边界校验，避免超出影像行列数导致的报错；实现多波段影像的循环读写，保留原影像投影、NoData 值、像素类型等元信息；修正地理坐标转像素坐标的取整逻辑，修复裁剪结果偏移问题，测试多组裁剪参数，工具可稳定输出有效裁剪影像。

优化影像裁剪小工具的实用性与健壮性，添加异常捕获机制，对文件打开失败、参数无效等情况给出明确提示；封装工具为独立函数，新增命令行参数解析，支持终端传参运行；多次测试不同格式影像，修复中文路径下的保存 bug，最终裁剪小工具功能完整、运行稳定，可满足日常影像裁剪需求。
